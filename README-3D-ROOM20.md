# Cập nhật kích thước phòng 3D 20m x 20m x 6m

Bản này quy ước `1 đơn vị Three.js = 1 mét thực tế`.

Đã chỉnh:

- Sàn: 20m x 20m.
- Tường/trần: cao 6m.
- Cửa giữa tường trước: rộng 3m, cao 2.7m.
- Giới hạn di chuyển an toàn: X/Z từ -8.6m đến 8.6m.
- 5 ngách trưng bày phân bổ lại theo phòng 20m x 20m.
- Điểm di chuyển xanh kéo ra xa tường để xoay 360 không bị lọt ra sau tường.
- Đèn, dầm trần và bóng đổ mở rộng theo kích thước phòng mới.

Cập nhật lên GitHub:

```bash
git add .
git commit -m "Resize 3D gallery room to 20x20x6 meters"
git push
```
