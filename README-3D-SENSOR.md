# Bản 3D Gallery có cảm biến điện thoại

Bản này bổ sung nút **Cảm biến** trong `index-3d.html`.

## Cách dùng trên điện thoại

1. Mở trang GitHub Pages bằng HTTPS, ví dụ:
   `https://phatvo.github.io/reactvr/index-3d.html`
2. Bấm **Cảm biến: Tắt** để bật cảm biến.
3. Nếu điện thoại hỏi quyền Motion/Orientation, chọn **Allow/Cho phép**.
4. Xoay điện thoại để nhìn quanh phòng.
5. Bấm **Hiệu chỉnh cảm biến** nếu góc nhìn bị lệch.

## Lưu ý

- iPhone/iPad thường chỉ cho dùng cảm biến khi website chạy bằng HTTPS và người dùng bấm nút để cấp quyền.
- Khi bật cảm biến, thao tác kéo chuột/xoay bằng OrbitControls sẽ tạm tắt để tránh xung đột.
- Vẫn có thể dùng cụm phím ảo hoặc các điểm di chuyển trên sàn để di chuyển.
